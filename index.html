1. Create a Google Sheet with sheets named: Students, Levels, Progress, Projects
2. Publish it: File → Share → Publish to web → Entire Document → CSV
3. Copy the Sheet ID from the URL and paste it in the SHEET_ID variable in this file's JavaScript section.
